# FarmMan Login Page

A modern and responsive login page built using **HTML**, **Tailwind CSS**, and **Font Awesome**. The project features a clean two-column layout with a vegetable-themed hero image and a professional login form.

## 🚀 Features

* Responsive design for desktop and mobile devices
* Modern login interface
* Tailwind CSS styling
* Font Awesome social media icons
* Clean and user-friendly layout
* Fast and lightweight

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* Font Awesome

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ankit-malikk/farmman-login.git
```

Navigate to the project folder:

```bash
cd farmman-login
```

Install dependencies:

```bash
npm install
```

Start Tailwind CSS build process:

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./dist/style.css --watch
```

Open `dist/index.html` in your browser.
